# 🏛️ Nedu | Global Knowledge Gateway

**Empowering the next generation through verified, accessible, and global education.**

---

## 🚀 The Mission
Nedu (formerly InfoFlow) is a digital NGO and "Test Lab" dedicated to democratizing knowledge. Our goal is to provide high-quality, 500-word "Micro-Courses" and verified articles to students worldwide. 

We are currently in **Phase 1: The 20-Author Challenge**, where we are recruiting contributors from 20 different countries to bring global perspectives to our platform.

## 🛠️ Tech Stack
This project is built with clean, semantic code to ensure accessibility and speed:
* **HTML5**: Structured for professional academic layouts.
* **CSS3**: Featuring a custom "Institute" aesthetic (Roboto & Fraunces fonts).
* **Flexbox/Grid**: For a responsive, "Sticky Footer" experience across all devices.
